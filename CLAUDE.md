# CLAUDE.md — INGCO Ukraine Frontend Quick Reference

This file provides a quick reference for development commands. For detailed coding guidelines, tech stack patterns, directory structures, and testing credentials, please refer to **[AGENTS.md](file:///f:/code/repos/fe-ingco/AGENTS.md)**.

## CLI Commands

- **Start dev server:** `npm run dev`
- **Production build:** `npm run build`
- **Post-build (sitemap):** `npm run postbuild`
- **Production server:** `npm run start`
- **Run lint check:** `npm run lint`
- **Prettier formatting:** `npm run prettier`
- **Prettier check:** `npm run prettier:check`
